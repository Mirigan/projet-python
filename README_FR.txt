Jeu de plateforme à 2 joueurs dont le but et de survive le plus longtemps en empechant les ennemis de sortir de la zone de jeu et en evitant de se faire toucher par ceux-ci.

/// ATTENTION : ce jeu est conçu pour une fenêtre de jeu de 1024x768. Merci de ne pas toucher a ces valeurs.

Lancer le jeu avec la syntaxe :
server.py 127.0.0.1 8888
client.py 127.0.0.1 8888

/// Règles du jeu :
¤ Une équipe de 2 joueurs
¤ 5 vies communes pour les 2 joueurs 
¤ -1 point de vie par ennemi heurté par un joueur
¤ -1 point de vie par ennemi sorti de la fenetre
¤ +1 point de score par ennemi touché
¤ La difficulté augmente progressivement en fonction du score

/// Difficultés :
Very Easy : 0-50
Easy : 50-100
Medium : 100-150
Hard : 150-200
Very Hard : >200

/// Objectif :
Survivre le plus longtemps possible et donc avoir le meilleur score possible


/// Commandes de jeu :
Fleche du haut : Saut
Fleche de droite : mouvement a droite
Fleche de gauche : mouvement a gauche
Barre d'espace : Tir
